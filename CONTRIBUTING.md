# Contributing to Hand Gesture Control System

Thank you for your interest in contributing! This document provides guidelines for contributing to the project.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/hand-gesture-control.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature-name`

## Development Setup

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run tests
npm test

# Build for production
npm run build
```

## Code Style

- Use ES6+ JavaScript features
- Follow existing code formatting
- Use meaningful variable and function names
- Add comments for complex logic
- Keep functions small and focused

## Adding New Gestures

See `DEVELOPER_GUIDE.md` for detailed instructions on adding new gestures.

## Testing

- Write unit tests for new features
- Ensure all tests pass before submitting PR
- Test on multiple browsers if possible
- Test with different lighting conditions

## Pull Request Process

1. Update documentation if needed
2. Add tests for new features
3. Ensure all tests pass
4. Update CHANGELOG.md
5. Submit PR with clear description

## Commit Messages

Use clear, descriptive commit messages:

```
feat: Add thumbs up gesture detector
fix: Resolve pinch detection false positives
docs: Update gesture guide with new examples
refactor: Simplify swipe detection logic
```

## Code of Conduct

- Be respectful and inclusive
- Provide constructive feedback
- Focus on the code, not the person
- Help others learn and grow

## Questions?

Open an issue or start a discussion on GitHub.

Thank you for contributing!
